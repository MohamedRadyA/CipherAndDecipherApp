# CipherAndDecipher

## The 10 cipher/decipher algorithms used are: 
1) Affine Cipher
2) Caeser Cipher
3) AtBash Cipher
4) Vignere Cipher
5) Baconian Cipher
6) Simple Substitution Cipher
7) Polybius Square Cipher
8) Morse Code Cipher
9) XOR Cipher
10) Rail-Fence Cipher
